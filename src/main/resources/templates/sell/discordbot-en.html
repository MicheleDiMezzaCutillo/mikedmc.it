What is a Custom Discord Bot?

A custom Discord bot is an application created just for you. It will have a name, profile picture, background, description, and personalized activity of your choice. It can have existing functions or commands that don’t exist and will be exclusive to you! For example, you can check out my projects or more specifically, the bots and the various embeds I've made, so you can see what might interest you.

Frequently Asked Questions

What language will the bot respond in?

You can choose the language. I mainly use English and Italian, but upon request, I can use other languages as well: just send me the texts you want to use.

What does "host" mean, and how can I keep my bot running 24/7?

Hosting a Discord bot means keeping it running so it can function continuously. To keep your bot active 24/7, you can either run it on your computer, leave it running on a virtual machine, or entrust hosting to me for a monthly fee.

What is a .jar file? Will I receive the source code?

A .jar file is a program created in Java that can be executed. After purchasing the bot, you will receive one of these files, which will contain the bot itself. Running it will start the bot. Of course, upon request, I can provide the source code along with the bot's .jar file.

What should I do if I encounter problems with the bot after delivery?

You can contact me, and I will fix the bugs as soon as possible. If the bot stops working due to API changes within 1 month of delivery, I will update it for free.

Do I need to share my Discord server with you?

Yes, for the initial installation and testing, I will need access to your server. Alternatively, I can create a temporary server to test and demonstrate the requested features.

Can you integrate other applications or websites?

Absolutely, using APIs or Web Scraping, it is possible to integrate applications and websites into your bot.

If the project is not easy to write or explain, can I talk to you directly?

Sure, I primarily use Discord and am always open to discussion to provide the best service possible in a clear manner.

Can you create slash commands, and can you use modules and buttons?

Yes, I can create slash commands for your bot. Additionally, I can use modules and buttons, which are often used in bots for ticket management and other interactive features.

I bought a bot and now I want to add new features. Can I do that?

Yes, if you have the source code (which can be requested at the time of bot delivery), I can add new features or modify existing ones for a fee based on the features to be added.

Can you modify an existing Discord bot?

I can modify a bot previously purchased from me if you have the source code. It is not possible to modify bots that I did not create.
