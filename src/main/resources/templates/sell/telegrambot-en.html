What is a Telegram Bot?

A Telegram Bot is an application created specifically for you. It will have a name, a profile picture, and a personalized bio of your choice. It can have existing functions or custom commands that are unique to you! For example, you can check out one of my projects that was commissioned to me.

Frequently Asked Questions

In which language will the bot respond?

You can choose the language. I mainly use English and Italian, but upon request, I can also use other languages: just send me the texts you want to use.

What does "hosting" mean, and how can I keep my bot active 24/7?

Hosting a Telegram bot means keeping it running so it can operate continuously. To keep your bot active 24/7, you can choose to run it on your computer, keep it running on a virtual machine, or have me handle the hosting for a monthly fee.

What is a .jar file? Will I receive the source code?

A .jar file is a program created in Java that can be executed. Upon purchasing the bot, you will receive one of these files, which will contain the bot itself. Running it will start the bot. Of course, upon request, I can also provide the source code in addition to the bot's .jar file.

What should I do if I encounter problems with the bot after delivery?

You can contact me, and I will resolve the bugs as soon as possible. If the bot stops working due to API changes within 1 month of delivery, I will update it for free.

Do I need to share any groups or contacts?

Yes, for initial installation and testing, it may be necessary. I will need access to your group if the bot is going to be in a group. I can also create a temporary group to test and demonstrate the required features.

Can you integrate other applications or websites?

Certainly, using APIs or web scraping, it is possible to integrate applications and websites into your bot.

If the project is difficult to describe or explain, can I speak to you directly?

Sure, I mainly use Discord and am always open to discussion to provide the best possible service clearly.

I bought a bot and now want to add new features. Can I do that?

Yes, if you have the source code (which can be requested at the time of bot delivery), I can add new features or modify existing ones for a fee based on the features to be added.

Can you modify an existing Telegram bot?

I can modify a bot previously purchased from me if you have the source code. It is not possible to modify bots that I did not create.
